# A simple Redis server implementation in Rust

## support commands

```shell
SET key value

GET key

DEL key [key ...]

HSET key field value

HGET key field

HMGET key field [field ...]

HKEYS key

HGETALL key

HDEL key field [field ...]

ECHO message

SADD key member [member ...]

SISMEMBER key member

SMEMBERS key

SREM key member [member ...]
```
